Midi

These Midi Files are also all on my DVD. Most of these were done some time ago (well before YouTube made videos possible), but some have been converted into videos.

[M] indicates that Markers (or annotations) have been embedded in the song. These mark and describe some examples of melodic and harmonic techniques used. They are best used with any version (even demo versions) of ‘Cakewalk’ or ‘Sonar’.

Many of these can be viewed as piano tutorial videos created with Synthasia on the youtube channel of Oriol Lopez Calle – Synthasia jazz piano tutorials
